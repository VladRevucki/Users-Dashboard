
## Установка и запуск

1. Установите зависимости:
   npm install

2. Запустите сервер разработки:
   npm run dev

3. Откройте [http://localhost:5173](http://localhost:5173) в вашем браузере.
# Панель мониторинга пользователей

Приложение для React TypeScript, отображающее панель мониторинга пользователей, полученную из API DummyJSON.

## Обзор

В этом проекте реализована пользовательская панель мониторинга, которая извлекает пользовательские данные из [DummyJSON Users API](https://dummyjson.com/docs/users) и отображает их в виде интерактивной карточки. Каждая карточка пользователя содержит важную информацию и может быть расширена для получения дополнительных сведений.

## Функции

- ** Отображение списка пользователей **: Отображение всех пользователей в адаптивной сетке
- ** Интерактивные карточки **: Нажмите, чтобы развернуть / свернуть дополнительную информацию о пользователях
- **Адаптивный дизайн **: Работает на настольных компьютерах и мобильных устройствах
- ** Безопасность ввода **: Полная поддержка TypeScript с соответствующими интерфейсами
- ** Состояния загрузки **: Отображает состояния загрузки и ошибки при извлечении данных
- **Поиск**: Фильтрация пользователей по имени, электронной почте или логину
- **Разбивка на страницы **: Навигация по большим спискам пользователей

### Почему React + TypeScript?

- **TypeScript**: Обеспечивает проверку типов во время компиляции, сокращает количество ошибок во время выполнения и улучшает работу разработчиков. Необходим для поддержки крупномасштабных приложений и обеспечения целостности данных при работе с внешними API.
- ** React**: Выбран из-за своей компонентной архитектуры, которая позволяет использовать повторно используемые компоненты пользовательского интерфейса, такие как карточка пользователя. Виртуальный DOM и перехватчики React упрощают управление состоянием и побочными эффектами (например, вызовами API).

### Почему Vite?

- **Быстрая разработка **: Vite обеспечивает молниеносную горячую замену модулей (HMR) и сокращает время сборки, повышая производительность разработчиков.
- ** Современный инструментарий **: Для переноса используется esbuild, который значительно быстрее, чем альтернативы, такие как Webpack.
- **Встроенные модули ES **: Поддерживают современные функции JavaScript "из коробки".

### Интеграция с API

- ** Собственный Fetch API **: Используется вместо библиотек, таких как Axios, для минимизации зависимостей и использования собственных возможностей браузера. Fetch API является современным, основанным на обещаниях и достаточным для данного варианта использования без увеличения размера пакета.
- **Обработка ошибок**: Реализует правильные состояния ошибок для корректной обработки сетевых сбоев.
